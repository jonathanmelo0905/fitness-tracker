# Comando: actualizar CLAUDE.md tras completar tarea

Actualiza el CLAUDE.md reflejando el estado actual del proyecto.

## Instrucciones

1. Lee el `CLAUDE.md` actual en la raíz del proyecto
2. Marca como `[x]` los ítems que se completaron en esta sesión
3. Si se crearon archivos nuevos, agrégalos a la sección de estructura que corresponda
4. Si se tomaron decisiones de arquitectura, agrégalas en la sección correspondiente
5. Actualiza la fase actual si se completó la fase entera
6. NO elimines ítems — solo márcalos como completados
7. Muestra un diff resumido de qué cambió en el archivo
